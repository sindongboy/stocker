# 전략: MA Crossover (이동평균 교차)

## 개요

단기 이동평균(MA5)과 장기 이동평균(MA20)의 교차를 매매 시그널로 사용하는 추세 추종 전략.

## 매수 조건 — 골든 크로스

- 전일: MA5 < MA20
- 당일: MA5 > MA20 (교차 발생)
- 거래량: 당일 거래량 ≥ 20일 평균 거래량의 80% 이상

## 매도 조건 — 데드 크로스

- 전일: MA5 > MA20
- 당일: MA5 < MA20 (교차 발생)

## 주문 파라미터

| 항목 | 값 |
|---|---|
| 주문 유형 | 지정가 (시장가 금지) |
| 매수 가격 | 현재가 (스냅샷 기준) |
| 주문 수량 | 가용 현금의 10% ÷ 현재가 (1주 단위 내림) |
| 최소 수량 | 1주 |
| 티어 | T1 (propose_order — 사용자 승인 필요) |

## 필터 (신호 무시 조건)

- MA5 또는 MA20이 None인 경우 (데이터 부족)
- 현재가가 0인 경우 (거래 정지)
- 교차 폭이 0.1% 미만인 경우 (노이즈)

## 분석 순서 (에이전트 가이드)

1. `search_knowledge("MA crossover")` — 이 카드 로드 확인
2. `get_portfolio()` — 가용 현금 확인
3. `get_market_snapshot(ticker)` — 현재가 조회
4. `get_indicators(ticker, ["MA5", "MA20"])` — 지표 계산
5. 교차 여부 판단
6. 시그널 있으면 `propose_order(...)` 호출
7. 반드시 `log_reasoning(...)` 으로 마무리
